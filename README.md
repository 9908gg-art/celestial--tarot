# MysticTarot: 線上奧秘之星塔羅牌占卜

這是一個專為塔羅牌愛好者與心靈探索者設計的**奢華暗金星芒、支援 3D 翻牌特效**的線上塔羅牌占卜工具。

專案使用純前端技術（HTML/CSS/JS）開發，可 **100% 免費託管於 GitHub Pages**，免除後端伺服器與資料庫維護，隨點隨算。

## 🌟 核心特色功能

1. **兩種古典經典牌陣 (Spreads)**：
   * **每日一牌 (今日指引)**：抽取單張牌卡，獲取今日的能量狀態與警示指引。
   * **時間之流 (三牌占卜)**：針對特定問題，剖析事情發展的「過去、現在、未來」脈絡。

2. **精緻的 3D 卡牌翻轉動畫**：
   * 使用純 CSS 3D 變換 (`perspective`、`transform-style: preserve-3d`、`backface-visibility`)。
   * 提供極具質感的牌背設計，點擊卡片時會有逼真的 3D 翻轉與正逆位旋轉特效。

3. **完整 22 張大阿爾克那 (Major Arcana) 資料庫**：
   * 包含愚者、魔術師、女祭司至世界等 22 張核心牌卡。
   * 內建精準的「正位」與「逆位」判斷，每張牌均有高達數百字的專業白話解密報告（包括能量焦點、個性優勢與具體指引）。

4. **奢華暗金視覺風格 (Tarot Celestial Theme)**：
   * 以神祕的紫羅蘭夜星與黃金壓紋線條作為視覺主調，營造出如同實體塔羅牌卡般的奢華儀式感。
   * 完美適配手機與平板，讓使用者隨時隨地都能進行占卜。

5. **100% 本地安全隱私保護**：
   * 洗牌與抽牌算法完全在使用者瀏覽器本地端運行，不儲存、不上傳使用者的問題或抽牌結果，保護個人隱私。

## 🚀 如何快速部署到 GitHub Pages？

1. 在 GitHub 上建立一個新的公開儲存庫（Public Repository），例如命名為：`celestial-tarot`。
2. 將本專案的四個檔案上傳至該儲存庫：
   * `index.html`
   * `style.css`
   * `app.js`
   * `README.md`
3. 進入儲存庫的 **Settings（設定）** ➡️ **Pages**：
   * 在 Branch 下方選擇 **`main`**，然後點選 **「Save」**。
4. 約 1 分鐘後，你的專屬塔羅牌網站即在網際網路上發布成功！
